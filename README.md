### 🔇 MUTE Control
- **Pin:** `GPIOB1`
- `High`  → Có nhạc  Low
- `Low` → Không có nhạc  

---

### 🎛 KEY Control
- **Pin:** `POWERKEY_ADC`
- Key 1 : 100 om
- Key 2 : 4K7 om
- Key 3 : 8K2 om
- Key 4 : 100K om
- Key 5 : 220K om
