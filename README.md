    - `NVM-TOOL-PC` hoặc  
    - `NVM-TOOL-MOBILE`
  - Thiết lập:
    - Class: `2.0 / 2.1 / 1.1`
    - Phím bấm
    - Biến trở

### 🔇 MUTE Control
- **Pin:** `GPIOB1`  
- `High` → Có nhạc  
- `Low` → Tắt nhạc  

---

### 🎛 KEY Control
- **Pin:** `POWERKEY_ADC`

| Key | Giá trị |
|-----|--------|
| Key 1 | 100Ω |
| Key 2 | 4.7KΩ |
| Key 3 | 8.2KΩ |
| Key 4 | 100KΩ |
| Key 5 | 220KΩ |
